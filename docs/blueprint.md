# **App Name**: Mindful Journey Landing

## Core Features:

- Appointment Scheduling: Direct WhatsApp integration for instant consultation scheduling.
- Carousel Testimonials: Smooth-transitioning carousel of real patient testimonials.
- Contact Form: Form collects user's name, email and WhatsApp number with a mandatory consent checkbox for data usage.
- Specialty Highlights: Displays the mental health specializations of Dra. Hajir Abdalla, specifically in anxiety, depression and mood disorders.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey calmness and trust.
- Background color: Very light blue (#E3F2FD), almost white, to create a clean and soothing atmosphere.
- Accent color: Light gray-blue (#90CAF9) to add depth without overwhelming the calm palette.
- Font pairing: 'Alegreya' (serif) for headlines, lending an elegant and intellectual feel, and 'PT Sans' (sans-serif) for body text to ensure readability.
- Use clean and minimalist icons for specialties and navigation, in the same soft blue as the primary color.
- Responsive design that adapts seamlessly to mobile, tablet, and desktop screens, following the structure of the reference site.
- Subtle, smooth transitions for the testimonial carousel to maintain a professional yet gentle user experience.
